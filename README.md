# 🏍️ MotoServiceCba | Gestión Integral

Plataforma interactiva para la cotización de servicios de mantenimiento preventivo y visualización de opciones de rectificación de motocicletas en Córdoba, Argentina.

---

## ✨ Características Principales

*   **Calculador Automatizado:** Cotiza al instante los costos de mantenimiento según cilindrada.
*   **Módulo de Reparación Pesada:** Menú interactivo para rectificación de cilindros, tapas, cigüeñales, embragues y cajas de velocidad.
*   **Diseño Premium y Responsivo:** Desarrollado con **Tailwind CSS** optimizado para celulares.
*   **Integración con WhatsApp:** Enlace directo para solicitar turnos fácilmente.

---

## 🛠️ Tecnologías

*   **HTML5 / Tailwind CSS** (Estilos modernos y rápidos).
*   **JavaScript (Vanilla)** (Lógica reactiva para el cálculo de precios).
*   **FontAwesome** (Iconografía profesional).

---

## 🚀 Cómo Ejecutar el Proyecto

1. Descarga o clona este repositorio.
2. Abre el archivo `index.html` en cualquier navegador web.
